# Sprint 2 — Inventario + Caja

**Duración estimada:** 1.5 semanas full-time / 3 semanas part-time.
**Pre-requisito:** Sprint 1 cerrado.

---

## HUs planeadas (a detallar al iniciar el sprint)

- HU-06: Registrar entrada de mercancía (admin).
- HU-07: Hacer ajuste manual de inventario con motivo (admin).
- HU-08: Ver kardex de un producto (admin).
- HU-09: Ver listado de productos con bajo stock.
- HU-10: Abrir caja con base inicial (cajero).
- HU-11: Registrar ingreso/egreso manual durante sesión.
- HU-12: Cerrar caja con cuadre (cajero) y reporte imprimible.

---

## Definición de "Hecho"
- Todas las HUs cumplen criterios de aceptación.
- Tests unitarios e integración para `inventory` y `cash-register`.
- Función `get_stock` testeada con casos extremos.
- RLS configurado.
- Documentación actualizada.
